# Getting started

You can use [ngrok](https://ngrok.com) to test payments locally

## Requirements

```
ruby 2.6.3
nodejs v10.16.0
yarn 1.21.1
postgresql 12.0
```

## Install 

```sh
 ./bin/setup
``` 


Take a look at the `.env` file. Replace `TODO: change me` with your own values.


## Run server

```
  bundle exec rails s

  go to: http://localhost:3000
```
